
# Neon Arena Quizzes 🎮⚡

## 📌 Overview
A real-time quiz platform with interactive UI and live chat features.

## 🚀 Features
- 🧠 Real-time quiz system
- 💬 Live chat using Supabase
- ⚡ Fast UI with React + Vite
- 🎨 Modern design using Tailwind & shadcn

## 🛠 Tech Stack
- Frontend: React (Vite)
- Backend: Supabase
- Styling: Tailwind CSS, shadcn UI

## 📂 Project Structure
src/ → main code
public/ → assets
supabase/ → backend config

## ⚙️ Installation
```bash
git clone <repo-link>
cd neon-arena-quizzes
npm install
npm run dev
